# restfulpy-client.js
Javascript client for restfulpy


## Setup

```bash
npm install
```

```bash
v.activate
mkvirtualenv --python=$(which python3) restfulpy-client.js

pip install restfulpy
```

## Running tests

```bash
npm run test
```

## Running tests, and watch for changes (development mode)

```bash
npm run dev
```


## Build

```bash
npm run build
```
